安装方式

方式一：CDN引入

开发环境版本
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js></script>
生产环境版本
<script src=https://cdn.jsdelivr.net/npm/vue></script>

方式二：下载和引入

https://cn.vuejs.org/v2/guide/installation.html#直接用-lt-script-gt-引入

方式三：使用npm进行安装

$ npm install vue
