v-bind动态绑定class(对象语法)

true时，此类添加到标签中；false时，不添加

没有v-bind的class和有v-bind的class会合并

v-bind动态绑定class(对象语法).html

注

{active: isActive, line: isLine}
可以使用方法代替，在methods中添加
getClasses: function() {
return {active: this.isActive, line: this.isLine};
}
再在之前的位置调用:
v-bind:class=”getClasses()”
