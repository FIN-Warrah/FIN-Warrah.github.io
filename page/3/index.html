v-model的使用和原理

发表于 2019-11-11 分类于 Vue.js学习笔记

表单绑定v-model

Vue通过v-model指令来实现表单元素和数据的双向绑定
双向绑定：改变其中一个，另一个也会随之改变

v-model的原理

v-model包含两个操作：

v-bind绑定一个value属性
v-on给当前元素绑定input事件

可以使用v-bind和v-on: input达到同样的效果

v-on: input监听是否填入内容
此监听函数参数为event，而通过event.target.value获取输入的值

或使用以下方法代替v-on:input传入函数

v-on:input=”message = $event.target.value”

数组中哪些方法是响应式的

发表于 2019-11-11 分类于 Vue.js学习笔记

数组中哪些方法是响应式的

push方法：再数组末尾添加元素
pop方法：从数组末尾弹出元素
shift方法：删除数组中的第一个元素
unshift方法：在数组最前面添加元素(可以是多个元素）
splice方法：删除元素/插入元素/替换元素

  • 删除元素：start, numbers：从start开始，删除numbers个元素
  • 删除元素：start：从start开始，删除所有元素
  • 替换元素：start, numbers, elements(numbers个)：从start开始，将numbers个元素替换为传入的elements
  • 插入元素：start, 0, elements(要插入多少个就传入多少个)
    sort方法：排序
    reverse方法：翻转

注意

通过索引修改数组元素不是响应式的,可以通过splice方法代替，或使用Vue.set

Vue的set方法

Vue.set(要修改的对象, 索引值, 修改后的值);

v-for绑定和不绑定key的区别

发表于 2019-11-11 分类于 Vue.js学习笔记

性能区别

若不绑定key，在数组中动态插入元素时会依次替换，性能低
若绑定key，会使用diff(difference)算法插入，性能高

组件的key属性

官方推荐我们再使用v-for时，给对应的元素或组件添加上一个:key属性

为什么要添加key属性

使用key来给每个节点做一个唯一标识
Diff算法就可以正确地识别此节点，找到正确的位置插入新的节点

v-show决定是否显示

使用v-show和使用v-if的区别

对于false

使用v-if：不会存在于DOM中
使用v-show：给元素增加了display: none 的行内属性

如何选择

当在显示和隐藏之前频繁切换时，使用v-show
若切换频率较低，一般使用v-if
