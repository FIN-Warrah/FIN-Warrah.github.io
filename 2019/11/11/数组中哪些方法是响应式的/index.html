数组中哪些方法是响应式的

发表于 2019-11-11 分类于 Vue.js学习笔记

数组中哪些方法是响应式的

push方法：再数组末尾添加元素
pop方法：从数组末尾弹出元素
shift方法：删除数组中的第一个元素
unshift方法：在数组最前面添加元素(可以是多个元素）
splice方法：删除元素/插入元素/替换元素

  • 删除元素：start, numbers：从start开始，删除numbers个元素
  • 删除元素：start：从start开始，删除所有元素
  • 替换元素：start, numbers, elements(numbers个)：从start开始，将numbers个元素替换为传入的elements
  • 插入元素：start, 0, elements(要插入多少个就传入多少个)
    sort方法：排序
    reverse方法：翻转

注意

通过索引修改数组元素不是响应式的,可以通过splice方法代替，或使用Vue.set
